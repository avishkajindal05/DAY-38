# Day 38: Word Embeddings & Similarity

This directory contains the notebook and scripts for the Day 38: Word Embeddings & Similarity assignment.

## Environment & Dependencies
- **Python Version:** 3.9+
- **Required Packages:**
  ```bash
  pip install pandas numpy scikit-learn gensim sentence-transformers scipy
  ```


## How to Run
1. Open up a terminal in this directory.
2. Ensure you have activated your preferred virtual environment.
3. Install the dependencies listed above.
4. Launch Jupyter Notebook or VS Code:
   ```bash
   jupyter notebook PART-A-B.ipynb
   ```
5. Click **Run All** in the notebook.

## Expected Output
Running the notebook end-to-end will produce:
- Word2Vec similarities (polysemy mapping), vector distances, and cosine similarities using BOW vs Embeddings.
